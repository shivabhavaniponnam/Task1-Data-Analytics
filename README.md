# Task 1: Data Immersion & Wrangling

📌 Objective
The objective of this task is to understand, explore, and prepare raw data for analysis by performing data cleaning and transformation.

📊 Dataset
The dataset used is the Superstore dataset obtained from Kaggle, which contains information about products, sales, and profits.

🔍 Steps Performed
* Explored the dataset using Python (Pandas)
* Checked for missing values and duplicates
* Performed data profiling to understand structure and data types
* Applied data cleaning techniques such as duplicate removal and handling missing values
* Converted date columns into proper format

📁 Files Included
* Sample - Superstore.csv (Original Dataset)
* cleaned_superstore.csv (Cleaned Dataset)
* task1.py (Python code for cleaning)
* data_dictionary.xlsx (Description of columns)

✅ Result
The dataset was analyzed and found to be well-structured. Data cleaning steps were applied and a final cleaned dataset was prepared for further analysis.
